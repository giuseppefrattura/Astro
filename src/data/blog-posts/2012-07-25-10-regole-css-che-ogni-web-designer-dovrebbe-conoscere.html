Eccovi dieci regole CSS che ogni web designer dovrebbe conoscere

@media

{% highlight ruby linenos %} @media screen and (max-width: 960px) { ... } {% endhighlight %} La regola media permette di specificare il layout di stampa del sito ed è anche alla base della creazione di pagine web responsive usando come attributo del campo media min-width si specificano le regole da applicare al sito quando è visualizzato su un display di determinate dimensioni.

Background-size

{% highlight ruby linenos %} body { background: url(image.jpg) no-repeat; background-size: 100%; } {% endhighlight %} Permette di riempire con un’immagine il 100% del body nel caso dell’esempio.

@font-face

{% highlight ruby linenos %} @font-face { font-family: Blackout; src: url("assests/blackout.ttf") format("truetype"); } {% endhighlight %} Permette di scegliere con facilità  i font da usare all’interno delle pagine selezionandoli manualmente da risorse gratuite o via google WebFonts o Typekit.

margin 0 auto;

{% highlight ruby linenos %} #container { margin: 0 auto; }{% endhighlight %} Serve a centrare qualsiasi blocco all’interno del blocco che a sua volta lo contiene.

Overflow:hidden

{% highlight ruby linenos %} .container { overflow: hidden; }{% endhighlight %} Un modo semplice per nascondere i contenuti che escono fuori da .container .

.clearfix

{% highlight ruby linenos %} .clearfix:after { content: "."; display: block; clear: both; visibility: hidden; line-height: 0; height: 0; }{% endhighlight %} Fa lo stesso lavoro di overflow : hidden quando quest’ultimo non funziona

Color: rgba();

{% highlight ruby linenos %} .btn { color: rgba(0,0,0,0.5); }{% endhighlight %} Con rbga si può impostare il colore di un elemento compresa la sua componente alpha che ne determina la trasparenza.

Input [type:”text”]

{% highlight ruby linenos %} input[type="text"] { width: 200px; }{% endhighlight %} Serve per modificare gli stili applicati ai campi delle form.

Trasform: rotate(30deg)

{% highlight ruby linenos %} .title { transform: rotate(30deg); }{% endhighlight %} Trasform permette di effettuare trasformazioni senza utilizzare javascript.

outline:none;

{% highlight ruby linenos %} A { outline:none; }{% endhighlight %} Permette di eliminare gli outline attorno agli elementi che preferisci.
