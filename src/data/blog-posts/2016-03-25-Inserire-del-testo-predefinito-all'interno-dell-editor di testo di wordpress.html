Quando si consegna un sito al cliente fatto con wordpress può essere una buona idea dera la possibilità nel momento in cui si crea un nuovo articolo o una nuova pagina di avere del testo già inserito in modo da agevolare il lavoro dello scrittore.
Ecco come fare per inserire un testo nell'editor di Wordpress senza l'ausilio di plugin.

Per farlo è necessario modificare il file functions.php del nostro template Wordpress inserendo in coda il seguente codice

{% highlight ruby linenos %} add_filter( 'default_content', 'pu_default_editor_content' ); function pu_default_editor_content( $content ) { global $post_type; switch( $post_type ) { case 'post': $content = 'Default content for blog posts.'; break; case 'page': $content = 'Default content for pages.'; break; } return $content; } {% endhighlight %}

In pratica se si crea una nuova pagina si inserirà un determinato testo, se si crea un nuovo articolo ci sarà un altro. Per altre info ti rimando al sito di wpRecipes (Inglese).
